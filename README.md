Installation
============

Se http://docs.casperjs.org/en/latest/installation.html.

1. Installera NodeJS (behöver PATH-variabeln uppdateras?)
2. Installera PhantomJS (glöm inte att uppdatera PATH-variabeln)
3. Installera CasperJS
    $ npm install
    
Kör
===

$ node_modules\casperjs\bin\casperjs.exe download.js --username=YOUR_ENTRE_USERNAME --password=YOUR_ENTRE_PASSWORD
